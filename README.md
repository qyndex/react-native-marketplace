# React Native Marketplace

Full marketplace with product catalog, search, cart, and checkout flow. State managed by Zustand; uses expo-router for deep-linking and Platform.select for responsive grid layouts.

**Stack:** React Native | **Complexity:** complex
**Tags:** `mobile`, `expo`, `react-native`, `marketplace`, `cart`, `zustand`, `ecommerce`

## Quick Start

```bash
npm install
npx expo start
```
